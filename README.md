<h1>Uputstvo za koriscenje</h1>

 - Skinite kod main.py
 - Skinite biblioteke preko komandi
```
pip install networkx
pip install numpy
```
 - Nakon toga promenjivu unos promenite u <b>listu susedstva grafa G pogodnu za kopiranje</b> sa sajta(poslednja linija pre polja za unos resenja a posle primera 10 sa desne strane.
 - Promenite <b>v</b> i <b>t</b> sa onima koji vam pisu u tekstu zadatka
 - Pokrenite program
 - Samo iskopirajte ono sto vam pise posle broja zadatka
Ako nadjete bilo kakav problem u kodu ili ne mozete da ga pokrenete slobodno me kontaktirajte
